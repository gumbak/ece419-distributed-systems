Assumptions
===========

- ZooKeeper is always running
- There are initially two JobTrackers and two FileServers (primary and backup)
- Clients will only submit jobs with alphanumeric input

Usage
=====

Make the components:

    	$ make

Run ZooKeeper Server:

    	$ sh runZookeeper.sh

Run JobTracker:

    	$ sh runJobTracker.sh

Run Client:

    	$ sh runClientDriver.sh

Run FileServer:

    	$ sh runFileServer.sh

Run Worker:

    	$ sh runWorker.sh
